# Academic Homepage (Jekyll Minima)

This repository is a ready-to-use academic homepage template using the **minima** theme (works natively on GitHub Pages).

## Quick start
1. Copy all files to your `username.github.io` repository.
2. Edit `_config.yml` (title, description, header pages).
3. Update `index.md` and the pages in the root folder.
4. Wait ~30 seconds and open `https://username.github.io`.
